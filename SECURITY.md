# Security Policy

MarketLab is a research project. Report security issues privately instead of opening a public issue.

## Reporting

Send a concise report that includes:

- what you found
- where it appears
- why it matters
- any steps to reproduce

## What To Report

- credential exposure
- supply-chain or dependency concerns
- data leakage
- unsafe command execution
- repository or release misconfiguration that creates a security risk

## Scope

This policy covers the repository, release artifacts, and automation workflows. It does not replace a formal incident-response program.
